### installation
1. clone the repo
2. `pip install -e .`

### Login from cli
1. get your apikey from our website
2.
```
project-x login --auth_token {your_apikey}
```


### Init your app
project-x init_app --project_name colab_test --mode model

### start server
`project-x start model --upload-url=http://localhost:3010/upload_file`
